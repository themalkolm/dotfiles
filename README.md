dotfiles
========

Various dotfiles & some initial configuration. Helps me to have same env
on different laptops.

how to install
==========

```sh
gem install homesick
homesick clone themalkolm/dotfiles
homesick rc dotfiles
homesick symlink
```

how to update
=============

```sh
homesick pull dotfiles
homesick rc dotfiles
homesick symlink
```
